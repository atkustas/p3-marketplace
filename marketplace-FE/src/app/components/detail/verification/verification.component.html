Available Environments

Hello World API operates in two separate environments: evaluation and production.

  • The evaluation environment returns mock data and is designed for testing the basic functionality of the API.

      API environment URL:

  • The production environment lets end-users retrieve data from real accounts connected via Hello World Link.
